# ReadyQt

**ReadyQt** est une bibliothèque de Widgets Qt pour le développement d'Interface Homme-Machine.  
C'est une bibliothèque développée en C++/Qt et maintenu par 
[Gerard KESSE](http://5.48.215.193:8855/presentation/ "Accédez à mon site web (ReadyDev)").

# Technologies

Le développement de la bibliothèque **ReadyQt** a nécessité les technologies suivantes :

* Langage : **C++**
* IDE : **Qt Creator**
* Outil de Construction : **QMake**
* Interface Homme-Machine : **Qt**
* Patron de Conception : **Singleton++**
* Cible : **Windows, UNIX**
